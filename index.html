MySQL忘记密码后重置密码( Mac )

​ 安装好MySQL以后，系统给了个默认的的密码，当时是第一次在Mac上安装，所以忽略了，后来登录MySQL的时候需要用到密码，但不知道密码，所以得修改这个密码了，然后就走上了不归路。

​ 这个过程是心酸的，网上的资料多如狗，关键是各有各的错法，尝试了半天没一个对的，在即将打破心理防线，要去翻MySQL文档的时候，成功了有没有。没有一个文章告诉我完整的答案，我是参考了好几个攻略，闭门造出来的车。给自己点赞。不多说了，一步一步跟我走。

  1. 关闭MySQL服务器

sudo /usr/local/mysql/support-files/mysql.server stop

或者

在偏好设置里点击MySQL然后将其关闭

  1. 进入MySQL的二进制命令目录下

cd /usr/local/mysql/bin

  1. 获取MySQL操作权限

cd /usr/local/mysql/bin

  1. 重启MySQL服务器

./mysqld_safe --skip-grant-tables

  1. 重开个终端

为了在中点可以使用MySQL的命令，需要配置一下，运行下面两条命令

  1. 输入mysql 进入mysql命令模式
  2. 输入use mysql 进入mysql数据库
  3. 输入flush privileges 获取修改密码的权限
  4. 输入set password for ‘root’@’localhost’=password(‘新密码’) 修改密码
  5. 恭喜你，密码修改成功
  6. 使用新密码登录

mysql -u root -p

回车后输入新密码

出现上面这个就说明成功登录MySQL
