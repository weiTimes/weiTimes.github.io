Mac 生成 ssh-key

用Git下载代码得步骤1.首先需要创建公钥，在命令行中输入： 1ssh-keygen -t rsa -C "ben@xxx.com" 然后输入路径，密码之类的，如果不输入，工程就会保存到根目录下面，密码也为空。 1~/.ssh/id_rsa是私钥，~/.ssh/id_rsa.pub是公钥 2.接着打开公钥： 1cat ~/.ssh/id_rsa.pub 这里会得到一段很长的编码: 1ssh-rsa AAAAB3Nza
